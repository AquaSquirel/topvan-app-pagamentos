# **App Name**: TopVan Manager

## Core Features:

- Dashboard: Displays a summary of total students, total received, and total pending payments. Monetary values are hidden by default, with an option to reveal/hide them.
- Student Management: Allows managing students divided by morning and night shifts, including adding, editing, and deleting student information.
- Payment Status: Enables marking student payments as 'Paid' or 'Pending' with visual status indicators.
- Institution Management: Provides a system to add and remove educational institutions, which appear in a dropdown menu when adding students.
- Reset Payments: Allows resetting all student payment statuses to 'Pending' for a new billing cycle.

## Style Guidelines:

- Background color: Dark background (#1a1a1a) to reduce eye strain.
- Primary color: Gold (#FFC107) to highlight main buttons, titles and icons.
- Accent color: Green for 'Paid' status, Red for 'Pending' status to give instant visual feedback.
- Body and headline font: 'Poppins', a geometric sans-serif font for a clean and modern look.
- Retractable lists for morning and evening classes to optimize space, especially on mobile devices.
- Use icons to indicate shift (sun for morning, moon for evening) and payment status (check mark, clock or warning).
